# EvoAtlas | Evolutionary Biology RAG System 
I developed EvoAtlas to streamline my learning journey in biology!

By leveraging a RAG (Retrieval-Augmented Generation) architecture,
the app processes and indexes core Wikipedia documentation across
15 essential biological themes, providing precise, context-aware
answers to complex evolutionary queries.
